## 🆘 Quick Troubleshooting

### ❌ "Firebase not initialized"
```bash
flutter clean
flutter pub get
flutter run
```

### ❌ "Permission denied"
Go to Firebase Console → Firestore → Rules → Set test mode

### ❌ App crashes on start
Check you ran: `flutter pub get`

**More help:** See `FIREBASE_QUICK_SETUP.md` (Common Issues)
